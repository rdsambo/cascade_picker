# cascade_picker

Cascading selectors

## Usage

#### 1\. Depend

Add this to you package's `pubspec.yaml` file:

```yaml
dependencies:
  cascade_picker: ^0.0.8
```

or

```yaml
cascade_picker:
    git:
      url: git://github.com/rdsambo/cascade_picker.git
```

#### 2\. Install

Run command:

```bash
$ flutter packages get
```

#### 3\. Import

Import in Dart code:

```dart
import 'package:cascade_picker/cascade_picker.dart';
```
